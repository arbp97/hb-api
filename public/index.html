St. Patrick Homebanking API

Repository Main page Documentation

> /auth - POST body header content response

Send a valid email and password - responds with sent email & access token (stp-token).

email: example@email.com
password: password app-token: ACCESS TOKEN application/json email: example@email.com
token: TOKEN error: error

> /account/find - GET body content header response

Send a valid cciCode OR accountNumber and token - responds with Account object.

cciCode: CCI accountNumber: "000-000000000-000" stp-token: TOKEN application/json Account error: error

> /account/transfer - PUT body content header response

Make a money transfer between two valid accounts. Origin and destiny refers to the CCI code.

origin: CCI
destiny: CCI
motive: some motive
amount: > 0 stp-token: TOKEN application/json status: success error: error

> /accounts/user - GET body content header response

Send a DNI OR email & valid token - responds with all Accounts with the same owner.

dni: DNI email: example@email.com stp-token: TOKEN application/json Account [] error: error

> /user/find - GET body content header response

Send a user's DNI & valid token - responds with User object.

dni: DNI stp-token: TOKEN application/json User error: error

> /user/update - PUT body content header response

Update individual fields (can be all fields specified except for DNI) of the User.

dni: DNI email: example@email.com name: John surname: Doe img: IMG state: "active" || "inactive" stp-token: TOKEN application/json User error: error

> /transaction/find - GET body content header response

Send a valid transaction ID & token - responds with Transaction object.

transactionId: >= 0 stp-token: TOKEN application/json Transaction error: error

> /transactions/account - GET body content header response

Send a valid account CCI & token - responds with all Transactions having this account as the origin or destiny.

cci: CCI stp-token: TOKEN application/json Transaction [] error: error
